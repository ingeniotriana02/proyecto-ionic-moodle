select

  Los selects son controles de formulario para seleccionar una opción u opciones de un conjunto de opciones similar a un elemento select nativo. Cuando un usuario toca el select, aparece un
  cuadro de diálogo con todas las opciones en una lista grande y fácil de seleccionar.
  Se debe usar una selección con elementos secundarios "ion-select-option". Si la opción secundaria no recibe un atributo de valor, su texto se utilizará como valor.
  Si el valor se establece en "ion-select", la opción seleccionada se elegirá en función de ese valor.

  1. SELECCIÓN UNICA

  Por defecto, el select permite al usuario seleccionar solo una opción. La interfaz de alerta presenta a los usuarios una lista de opciones con estilo de botón de radio. El valor del componente
  seleccionado recibe el valor de la opción selecccionada.
