items

Los items son elementos que pueden contener texto, iconos, avatares, imagenes, entradas (inputs) y cualquier otro elemento nativo o personalizado. Generalmente se colocan en una lista con otros elementos. Los elementos se pueden despizar, eliminar, reordenar, editar y más.

1.USO BÁSICO

Los items alinean el texto a la izquierda y agregan puntos suspensivos cuando el texto es mas ancho que el elemento. Podemos modificar este comportamiento utilizando las utilidades de CSS proporcionadas por Ionic Framework, como usar ".ion-text-wrap"en el siguiente ejemplo. 2. ITEMS CLICKABLE

Se considera que se puede hacer click en un items si tiene un conjunto de propiedades href o button. Los items en los que se puede hacer click tiene algunas diferencias visuales que indican que se puede interactuar con ellos como por ejemplo una flecha o que está más "visible" que otro indicando que el que esta menos visible no pueda ser clickable.
